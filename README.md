# E-Portfolio

Hello World!

## Dependencies

1. **Framer-Motion**
    ```bash
    npm install framer-motion
    ```

2. **Router Dom**
    ```bash
    npm install react-router-dom
    ```

3. **Material Icons**
    ```bash
    npm install @mui/icons-material @mui/material @emotion/styled @emotion/react
    ```

## Dev Notes

To start:
```bash
cd my-portfolio
npm start
```
